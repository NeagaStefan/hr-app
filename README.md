# HR App

A full-stack HR application with a Java Spring Boot backend and a React frontend.

## Prerequisites
- Java 21 or newer
- Maven
- Node.js 22 & npm

## Project Structure
- `hr-app-backend/`: Spring Boot backend
- `hr-app-frontend/`: React frontend

## Setup

### Backend
```bash
cd hr-app-backend
mvn clean install
mvn spring-boot:run
```

### Frontend
```bash
cd hr-app-frontend
npm install
npm start
```
